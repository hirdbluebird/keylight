General Layout

Keylight is based on groups of content and content containers.

If you take a closer look at the code sample below, you'll see that a .content-container can reside alone or within a .container-group. That's because .container-group elements have background images with the parallax effect and the image needs to go under two .content-container elements, rather than one.

That's the basic structure of the template:

<body>
<div class="nav-container"></div>
<div id="section1" class="section container-group">
<div class="subheader-container"></div>
<div class="content-container"></div>
</div>

<div id="section2" class="section content-container"></div>
</body>

Changing colors

Keylight is built on LESS which makes it fairly easy to change the main colors of the template.

Just open css/main.less and change the variables you need. They start at the beginning of the document and are marked as follows:

@color1: #18c08f;
@color2: #9a7192;
@grey: #e2e2e2;

After changing the values, you have to compile the LESS file. You can use the Crunch App for this, which can be downloaded here: http://crunchapp.net/

You can also use LESS directly on the template by uncommenting these two lines in the head section:

<!-- <link rel="stylesheet/less" type="text/css" href="css/main.less"> -->

and

<!-- <script src="js/less-1.4.1.min.js"></script> -->

Background Images

Background Images can either use the Parallax plugin, or just be fixed to the background.

They are added inline within the HTML on either a .container-group or a .content-container element. You can also combine these two and add an additional .color1 or .color2 class to the child .content-container element to have a color faded background image. Here's the example:

<div id="video" class="section container-group clearfix" style="background-image: url(img/samples/background-2.jpg)">
<div class="content-container color1">
<div class="wrapper padding-large clearfix center-text video">
</div> <!-- .wrapper -->
</div> <!-- .content-container -->
</div>

According to the class you add to .content-container or .container-group, the background color can be default(dark grey, when you add no class), white(.white), grey(.grey), color1(.color1) or color2(.color2). This will also define the color of the text in these elements. You can change text color with LESS variables.

Note: Use the included PSD when you make background images. It has a faded black overlay and the recommended dimensions (1950x1200).

Slider & Tabs

All carousels in this template are made with CarouFredsel - http://caroufredsel.dev7studios.com/

Please take a closer look at this plugin's configuration documentation if you want to modify some of the carousels.

The plugin is used on:

  • The portfolio section - uses the whole width of the window and lowers the number of items visible according to the width of the window
  • The quotes sections - a single item carousel
  • The single portfolio item - again a single item carousel, but with boxed image

Knob Elements

We're using a modified version of jQuery Knob for the knob elements at the top of the page. http://anthonyterrien.com/knob/

Because this plugin is not responsive, we're not changing the size of the knobs when resolution gets lower, just rearranging them. If you want to add an additional knob, you'll have to edit a bit the layout, use fourths instead of thirds and retest for mobile.

Setting up the Map

This template is using the standart Google Maps API.

It's also using a customized color skin on top. You can find a lot of samples and code snippets here: https://developers.google.com/maps/documentation/javascript/examples/

If you just want to change the address, open js/main.js and navigate to /* Map */. If you inspect the code below you'll find a line (center: new google.maps.LatLng(48.855, 2.341418),) - that is where the screen will be centered. A little bit below is this line (var myLatLng = new google.maps.LatLng(48.855, 2.341418);) - that's where the pin will be placed.

You can find out your coordinates by opening Google Maps and observing the URL in the address bar.

Adding a Video

Adding a video is pretty easy.

First of all don't forget to set a .video class on the .wrapper element, because otherwise it won't be responsive. Then just go to Youtube/Vimeo and choose to share/embed a video. Copy the code within the .wrapper element.

Dribbble Feed

There's a bonus Dribbble feed plugin included in the template.

You can find the Dribbble feed version of the template if you open index-dribbble.html from the download package. It's not included in the original preview because it slows down the whole template when it's combined with all of the other elements. If you replace the portfolio with it, though, you can get some pretty good results. Click the Dribbble button from the top navigation to see it.

To change the username, open js/main.js and under /* Dribble list */ change boyankostov to the desired username.

Setting up the Contact Form

The contact form is based on the jQuery Validation Plugin - https://github.com/jzaefferer/jquery-validation and the jQuery Form plugin http://jquery.malsup.com/form/.

Open process.php to change the email. If you want to use SMTP instead of php mail() you'll have to rely on PHP Mailer, Swift Mailer or another package and set it up manually.
